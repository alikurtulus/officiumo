Install Cypress

```bash
npm install cypress --save-dev
```

Install all npm packages

```bash
npm install
```
Source shell file

```bash
source ~/pp-cypress.sh
```

Run phoenix portal locally in another terminal

```bash
npm run dev
```

Run cypress

```bash
npx cypress open
```
